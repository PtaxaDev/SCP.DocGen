## 📋 О проекте

Визуальный редактор для создания документов в стиле SCP Foundation. Позволяет быстро и удобно оформлять отчёты об объектах, экспериментах и полевых ситуациях с полным соблюдением лора.

Проект выполнен в аутентичном стиле документов Фонда — строгая чёрно-белая эстетика с цветовыми акцентами, зависящими от класса объекта.

---

## 🚀 Возможности

### 📄 Создание документов
- Формирование документов в официальном стиле Фонда
- Автоматическое обновление метаданных (объект, класс, дата, автор)
- Динамическая смена цветовой схемы при выборе класса объекта
- Защита заголовков и метаданных от случайного редактирования

### ✏️ Редактирование
- Форматирование текста: жирный, курсив
- Создание редакций ███
- Поддержка пользовательских категорий разделов
- Водяные знаки "SCP FOUNDATION - OFFICIAL DOCUMENT"

### 🖼️ Работа с изображениями
- Загрузка изображений прямо в документ
- Библиотека загруженных изображений
- Автоматическая подпись под фото
- Просмотр в увеличенном размере по клику

### 🗺️ Полевые коды
- Набор визуальных кодов для экстренных ситуаций
- 5 категорий:
  - The Foundation Has Been Here
  - Hostile Being
  - It Is Not Human
  - Team Member Killed
  - Alternate Reality Entered
- Вставка через короткие теги
- Автоматическая замена на иконки в документе

### 📥 Экспорт
- Сохранение в PDF одной кнопкой
- Готово для печати и архивации
- Сохранение всего форматирования и изображений

## 🎯 Для кого этот проект

- **Авторов SCP-объектов** — для быстрого оформления документации
- **Игроков в RPG** — для создания улик, отчётов и артефактов
- **Фанатов вселенной** — для генерации документов по любимым объектам
- **Ролевых сообществ** — для стандартизации отчётов

---

## 🔧 Как использовать

1. Выберите класс объекта — цветовая схема изменится автоматически
2. Заполните метаданные (номер объекта, дата, автор)
3. Добавьте необходимые категории разделов
4. Напишите текст, используя форматирование
5. Вставьте изображения через теги
6. При необходимости добавьте полевые коды
7. Нажмите "Скачать PDF" для экспорта
